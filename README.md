## Integration

### Configuring gradle

For the integration of YandexMobileAds SDK need:
##### 1. compile AppMetrica SDK:

```sh
compile 'com.yandex.android:mobmetricalib:3.5.3'
```

##### 2. compile YandexMobileAds SDK:

```sh
compile 'com.yandex.android:mobileads:2.91'
```
